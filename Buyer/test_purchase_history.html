📋 API Endpoint Details

Endpoint: GET /api/buyer/purchase-history/

Query Parameters:
- status: all, pending, approved, completed, rejected
- date_from: YYYY-MM-DD format
- date_to: YYYY-MM-DD format
- page: Page number (default: 1)
- per_page: Items per page (default: 20)
Response Format:
{
  "success": true,
  "purchase_history": [...],
  "pagination": {...},
  "summary": {...}
}
